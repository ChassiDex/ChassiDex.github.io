Welcome to ChassiDex


ChassiDex is a database of host organisms used in synthetic biology. Under each host organism, all the information a synthetic biologist would need to work with that organism is listed. We also work on software tools to make working with different hosts easier.

Scroll below to view our list of organisms


We have also made browser based tools, for generating codon tables from genome data of any organism (cute.chassidex.org), and to carry out codon optimization for organisms on ChassiDex (copter.chassidex.org). However, please note that these are still in beta stage.


Also, check out our icon font for drawing synbio circuits with ease. comicsyns.chassidex.org
